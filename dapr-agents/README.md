# Dapr Agents Quickstarts

| Quickstart | Description |
|---|---|
| [durable-agent](./durable-agent) | A single durable agent that acts as a travel assistant, searching for flights and hotels with persistent state and conversation memory. |
| [multi-agent-workflow](./multi-agent-workflow) | A workflow-orchestrated multi-agent system where a predefined workflow coordinates triage and expert agents for customer support. |
| [multi-agent-orchestrator](./multi-agent-orchestrator) | An LLM-powered orchestrator agent that dynamically coordinates triage and expert agents for customer support, deciding at runtime which agents to delegate to. |
